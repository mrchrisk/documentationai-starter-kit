---
title: Data Import Errors
description: Fix common import issues in Market Trak Pro.
---

## Symptoms

- Data file fails to upload
- Error: "Invalid format" or "Import failed"

## Quick Fix

<Steps>
  <Step title="Check file format">
    ````tip
    Only CSV and XLSX files are accepted.
    ````
  </Step>
  <Step title="Remove empty rows/columns">
    Delete all blank rows and save again.
  </Step>
  <Step title="Retry import">
    Re-upload the cleaned file via the Dashboard.
  </Step>
</Steps>
