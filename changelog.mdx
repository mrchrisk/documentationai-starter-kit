## Check Your Version

<Steps>
  <Step title="Access Settings" icon="settings">
    Log in to `https://dashboard.marketrak.pro` and navigate to Account > Settings.
  </Step>
  <Step title="View Updates" icon="info">
    Scroll to the Version section to see your current release and changelog link.
  </Step>
  <Step title="Enable Notifications" icon="bell">
    Turn on email alerts for new major releases.
  </Step>
</Steps>

<codegroup tabs="Dashboard CLI">
````bash
# Check version via dashboard API
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.example.com/v1/version
````
````javascript
// Fetch latest version info
const response = await fetch('https://api.example.com/v1/version', {
  headers: { Authorization: `Bearer ${YOUR_TOKEN}` }
});
const version = await response.json();
console.log(version.current);
````
</codegroup>
